#  Banking System

A command-line Banking System built with **Python**, **SQLite**, and **Object-Oriented Programming (OOP)**.

The project simulates a simple banking application where users can create accounts, perform transactions, transfer money, and manage their banking information while storing all data in an SQLite database.

---

##  Features

* Create Customers
* Create Savings Accounts
* Create Checking Accounts
* Automatically generate account numbers (GR0001, GR0002, ...)
* View all accounts
* Deposit money
* Withdraw money
* Transfer money between accounts
* View account balance
* View transaction history
* Apply interest to savings accounts
* Close accounts (only when balance is €0.00)

---

---

##  Database Design

The application uses SQLite with the following tables:

* customers
* accounts
* savings_accounts
* checking_accounts
* transactions

Relationships are enforced using **Foreign Keys**.
